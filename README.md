# Olivine

A local-first CLI spaced repetition tool for Obsidian vaults.

## Status

Early development. Core tooling is being built.

## Development

```bash
npm install
npm run build
npm test
```

## License
MIT

## Usage

### Initialize a vault

```bash
olivine init /path/to/obsidian-vault
```

### Scan vault for notes

```bash
olivine scan /path/to/obsidian-vault
```
